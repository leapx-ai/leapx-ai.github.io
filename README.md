This is a Jekyll theme starter based on the [**Chirpy**][chirpy] theme.

## Quick Start

Follow the [**Getting Started**][chirpy-getting-started] guide to complete the setup of your site and make it available on the web.

Join the [**Chirpy community**][chirpy-discussions] to ask questions, share ideas, and get support.

Found a bug or have a feature request? Please [open an issue][chirpy-issues] or submit a pull request on the [theme repository][chirpy].
